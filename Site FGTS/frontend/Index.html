Ainda tem dúvidas?

Respondemos as perguntas mais comuns para você.

Preciso ter o nome limpo para antecipar?

Não! Como a garantia da operação é o seu próprio saldo do FGTS, a antecipação está disponível até mesmo para negativados. É uma ótima oportunidade para quitar dívidas.

Como funciona o pagamento?

Você não paga nenhuma parcela mensal. O pagamento é feito anualmente, com o débito automático da parcela do Saque-Aniversário diretamente do seu saldo do FGTS. Você não precisa se preocupar com boletos.

O que é o Saque-Aniversário?

É uma modalidade opcional do FGTS que permite sacar uma parte do saldo anualmente, no mês do seu aniversário. Para antecipar, você precisa ter aderido a essa modalidade no aplicativo do FGTS.
